# Magic Mod (Fabric 1.20.1)

Сделан рабочий прототип мода на магию с интерфейсом, крафтом 9x9, переключением заклинаний и HUD маны.

## Что работает сейчас
- `spell_scroll` — свиток конкретного заклинания.
- `blank_scroll` — пустой свиток-заготовка для **исследования 9x9**.
- `research_scroll` — открывает интерфейс биндов заклинаний.
- После изучения свитка заклинание сохраняется у игрока в NBT навсегда.
- Уровень магии, опыт и мана (с регенерацией).
- Требование уровня для каста убрано (важна только мана/изучение/бинд).
- Реальные эффекты заклинаний:
  - `fireball` — взрыв в точке прицеливания;
  - `blink` — телепорт вперёд;
  - `ice_spike` — замедление/ослабление существ рядом.

## Интерфейсы
1) `research_scroll` (ПКМ) — окно биндов:
- выбор изученного заклинания;
- `Бинд N` для слотов 1..9;
- `Очистить` для удаления бинда.

2) `blank_scroll` (ПКМ) — **поле исследования 9x9**:
- нажимай по клеткам, чтобы менять символ (`_`, `a`, `b`, `c`, `d`);
- `Синтез` отправляет паттерн 81 символ на сервер;
- если совпал рецепт, создается `spell_scroll` и расходуется `blank_scroll`.

## Управление кастом
- `<` и `>` — переключение активного слота среди забинденных.
- `R` — каст из выбранного слота.

## HUD игрока
На HUD отображается:
- текущая мана/максимум и уровень магии;
- активный слот и бинды для слотов 1..9;
- подсказка по клавишам `<`, `>`, `R`.
