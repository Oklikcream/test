# Magic Mod (готовая заготовка под Fabric 1.20.1)

Реализован рабочий MVP мода на магию для:
- Minecraft **1.20.1**
- Fabric Loader **0.15+**
- Java **17**

## Что уже работает
- Предмет `spell_scroll` (свиток заклинания).
- Предмет `research_scroll` (свиток исследования / создания заклинаний).
- Исследование рецептов на базе **9x9** паттерна (81 символ).
  - совпадение рецепта => создается свиток заклинания;
  - особый рецепт => магический взрыв;
  - иначе ничего.
- Перманентное изучение заклинаний (сохранение в NBT игрока).
- Уровень магии, мана, опыт.
  - опыт дается за крафт **новых** заклинаний и за каст.
  - максимальная мана растет вместе с уровнем.
- Система биндов заклинаний на слоты `1..9`.
- Реализация эффектов заклинаний:
  - `fireball` — взрыв в точке прицеливания,
  - `blink` — телепорт вперед,
  - `ice_spike` — массовое замедление рядом с игроком.
- Пассивная регенерация маны на сервере (тик-обновление профиля).

## Команды
- `/magic give_scroll <spellId>` — выдать свиток.
- `/magic level` — текущий уровень, мана и опыт.
- `/magic spells` — список изученных заклинаний.
- `/magic bind <1..9> <spellId>` — привязать изученное заклинание к слоту.
- `/magic unbind <1..9>` — очистить слот.
- `/magic bindings` — показать все слоты биндов.
- `/magic bind_menu` — командное «меню» по шагам для настройки биндов.
- `/magic ui` — чат-интерфейс с кликабельными кнопками для бинда заклинаний в слоты.
- `/magic cast <1..9>` — применить заклинание из слота с детальной причиной ошибки, если каст не удался.
- `/magic research <81 символ>` — исследование паттерна (модель поля 9x9).

Демо-рецепты:
- `a...a` (81 `a`) => свиток `fireball`
- `c...c` (81 `c`) => свиток `blink`
- `d...d` (81 `d`) => свиток `ice_spike`
- `b...b` (81 `b`) => магический взрыв

## Быстрый запуск
1. Установи Java 17.
2. В папке проекта выполни:
   - Linux/Mac: `./gradlew runServer`
   - Windows: `gradlew.bat runServer`
3. Зайди на сервер и используй команды выше.
