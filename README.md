# Magic Mod (Fabric 1.20.1)

Сделан рабочий прототип мода на магию с интерфейсом биндов, крафтом через `blank_scroll` и HUD.

## Что работает сейчас
- `spell_scroll` — свиток конкретного заклинания.
- `blank_scroll` — предмет для создания заклинаний через **простую сетку 5x5**.
- `research_scroll` — открывает интерфейс биндов заклинаний.
- После применения `spell_scroll` заклинание изучается навсегда (NBT игрока).
- Уровень магии, опыт и мана (с регенерацией).
- Каст не требует уровня: важны только бинды, изучение и мана.

## Создание заклинаний (blank_scroll)
- ПКМ по `blank_scroll` открывает экран 5x5.
- Клик по клетке: `.` -> `#` (закрашивание), повторный клик снимает заливку.
- Кнопка `Синтез` отправляет текущий 5x5 паттерн на сервер.
- На каждую попытку тратится 1 `blank_scroll` (кроме creative).
- Результат попытки: `spell_scroll` / взрыв / ничего.

## Интерфейс биндов
`research_scroll` открывает окно биндов:
- Выбор изученного заклинания.
- `Бинд N` для слотов 1..9.
- `Очистить` для удаления бинда.

## Управление кастом
- `<` и `>` — переключение активного слота.
- `R` — каст из выбранного слота.

## HUD игрока
На HUD отображается:
- мана, уровень и опыт (`exp/currentToNext`);
- активный слот и бинды 1..9;
- подсказка по клавишам `<`, `>`, `R`.
