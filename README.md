# Magic Mod Prototype

Прототип логики мода на магию для Minecraft:
- свитки заклинаний;
- исследовательский свиток с сеткой 9x9;
- постоянное изучение заклинаний;
- меню биндов на клавиши;
- уровень магии;
- система маны.

Это backend-логика (чистый Java), которую можно подключить к Fabric/Forge UI и игровым событиям.
