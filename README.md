# Magic Mod Prototype

Это **не готовый mod jar**, а прототип геймплейной логики (чистый Java), который нужно подключить к Minecraft API (Fabric/Forge/NeoForge).

## На какую версию это сейчас?
Сейчас проект **не привязан** к версии Minecraft: в нем нет зависимостей на `net.minecraft`.

## Рекомендуемая версия для реального мода
Рекомендую стартовать с:
- **Minecraft 1.20.1**
- **Fabric Loader 0.15+**
- **Fabric API** под 1.20.1
- **Java 17**

Почему 1.20.1: это очень распространенная и стабильная версия для модпаков, с большим количеством примеров и библиотек.

## Что уже реализовано в прототипе
- Свитки заклинаний.
- Исследовательский крафт 9x9 (результат: свиток / взрыв / ничего).
- Постоянное изучение заклинаний игроком.
- Бинды заклинаний на клавиши (модель данных).
- Уровень магии и опыт.
- Мана, растущая с уровнем.
