# Magic Mod (Fabric 1.20.1)

Сделан рабочий прототип мода на магию с интерфейсом, клавишами каста и HUD маны.

## Что работает сейчас
- `spell_scroll` — свиток конкретного заклинания.
- `blank_scroll` — **пустой свиток**, который при ПКМ записывает изученное заклинание в `spell_scroll`.
- `research_scroll` — открывает интерфейс магии.
- После изучения свитка заклинание сохраняется у игрока в NBT навсегда.
- Уровень магии, опыт и мана (с регенерацией).
- Реальные эффекты заклинаний:
  - `fireball` — взрыв в точке прицеливания;
  - `blink` — телепорт вперёд;
  - `ice_spike` — замедление/ослабление существ рядом.

## Интерфейс
ПКМ по `research_scroll` открывает окно **Magic Interface**, где есть кнопки:
- выбор изученного заклинания;
- `Бинд N` для слотов 1..9;
- `Очистить` для удаления бинда.

Каст заклинаний — **только с клавиатуры** по клавишам `1..9`.

## HUD игрока
На HUD отображается:
- текущая мана/максимум и уровень магии;
- строка биндов для слотов 1..9.

## Замечание
Старый командный UX (`/magic ...`) убран: управление перенесено в интерфейс и сетевые пакеты.
