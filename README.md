# Magic Mod (Fabric 1.20.1)

Сделан рабочий прототип мода на магию с интерфейсом, крафтом 5x5 (через 9 селекторов), переключением заклинаний и HUD.

## Что работает сейчас
- `spell_scroll` — свиток конкретного заклинания.
- `blank_scroll` — расходник для синтеза (1 шт на попытку).
- `research_scroll` — открывает интерфейс биндов заклинаний.
- После применения `spell_scroll` заклинание изучается навсегда (NBT игрока).
- Уровень магии, опыт и мана (с регенерацией).
- Каст не требует уровня: важны только бинды, изучение и мана.

## Интерфейс исследования
`blank_scroll` открывает окно с **9 выпадающими списками** (3x3).
- Каждый список выбирает руну: `Пусто`, `Fireball`, `Chaos`, `Blink`, `Ice Spike`.
- По `Синтез` создается скрытый паттерн 5x5 (25 символов) и отправляется на сервер.
- Результат: `spell_scroll` / взрыв / ничего.
- `blank_scroll` в любом случае расходуется на попытку (кроме креатива).

## Интерфейс биндов
`research_scroll` открывает окно биндов.
- Выбор изученного заклинания.
- `Бинд N` для слотов 1..9.
- `Очистить` для удаления бинда.

## Управление кастом
- `<` и `>` — переключение активного слота.
- `R` — каст из выбранного слота.

## HUD игрока
На HUD отображается:
- мана, уровень и опыт (`exp/currentToNext`);
- активный слот и бинды 1..9;
- подсказка по клавишам `<`, `>`, `R`.
