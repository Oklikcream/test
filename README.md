# Magic Mod (готовая заготовка под Fabric 1.20.1)

Окей, сделал максимально просто.

Теперь это **реальная заготовка мода** под:
- Minecraft **1.20.1**
- Fabric Loader **0.15+**
- Java **17**

## Что уже работает
- Предмет `spell_scroll` (свиток заклинания).
- Перманентное изучение заклинаний (сохранение в NBT игрока).
- Уровень магии, мана, опыт.
- Команда выдачи свитка: `/magic give_scroll <spellId>`
- Команда статуса: `/magic level`
- Команда исследования 9x9-паттерна: `/magic research <81 символ>`
  - `a...a` (81 `a`) => свиток `fireball`
  - `b...b` (81 `b`) => магический взрыв
  - иначе ничего

## Быстрый запуск
1. Установи Java 17.
2. В папке проекта выполни:
   - Linux/Mac: `./gradlew runServer`
   - Windows: `gradlew.bat runServer`
3. Зайди на сервер и используй команды из блока выше.
